# Panduan Pengguna (User Guide) Al-Manhaj

Selamat datang di Al-Manhaj. Sebuah habitat berpikir bebas distraksi.

## Memulai Zettelkasten
Zettelkasten bermakna kotak catatan. Anda diminta untuk mencatat sesuatu seringkas mungkin, namun menautkannya dengan konsep lain agar tidak menguap.

### 1. Mode Inkubasi
Sebelum gagasan matang, tangkap mereka pada menu **Inkubasi**. 
- Tekan fitur **Jernihkan Silogisme (AI)** untuk mengoptimalkan kata-kata yang terserak.
- Inkubasi adalah "Draf".

### 2. Tautan Dua Arah (Bidirectional)
Saat menulis editor di sebuah dokumen, ketik sintaks ganda:
`[[Pendidikan Akhlak]]`
Aplikasi akan mendeteksi apakah dokumen "Pendidikan Akhlak" ada. Jika ada, mereka saling bertaut. Jika tidak, aplikasi menyarankan Anda membuat dokumen baru.

### 3. Merawat "Catatan Yatim"
Jangan biarkan kerja keras Anda hilang. 
1. Masuk ke menu **Catatan Utama** (Indeks).
2. Tekan menu pop-up opsi (Penyesuaian).
3. Pilih penyaringan **"Yatim (0 Tautan)"**.
4. Semua pemikiran yang Anda lupakan karena belum tertaut akan muncul. Segera kaitkan mereka ke buku / kurikulum, atau kaitkan ke pemikiran permanen lainnya.

### 4. Ekspor Anti-Borgol (Data Liberation)
1. Pergi ke menu Index Catatan Utama.
2. Temukan ikon / tombol "Ekspor .md".
3. Al-Manhaj akan menyusun kepingan pengetahuan Anda ke dalam berkas ZIP utuh, berisi catatan Markdown murni.

## Arsitek Kurikulum
Manfaatkan menu Mutaba'ah untuk mengatur proyek riset besar (Buku ber-Bab, Target Hafalan, dsb) secara otomatis menggunakan AI Engine. AI akan memecah target Anda per-titik, dan Anda cukup menandai "Selesai" untuk memperbarui bilah *Progress*.

Selamat berpikir dan merekonstruksi peradaban dari satu paragraf ke paragraf lain!
